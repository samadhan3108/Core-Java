# Core Java

A collection of Core Java programs, concepts, and coding exercises designed to strengthen Java fundamentals. This repository covers essential topics from beginner to intermediate level with clean code, practical examples, and problem-solving approaches.

## Topics Covered




* Java Basics
* Variables & Data Types
* Operators
* Input & Output
* Conditional Statements
* Loops
* Methods
* Arrays
* Strings
* Object-Oriented Programming (OOP)
* Constructors
* Inheritance
* Polymorphism
* Abstraction
* Encapsulation
* Interfaces
* Packages
* Exception Handling
* Collections Framework
* File Handling
* Multithreading
* Wrapper Classes
* Generics
* Java 8 Features (Lambda Expressions & Streams)
* Practice Programs
* 
* Coding Challenges
